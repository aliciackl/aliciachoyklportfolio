Real Estate Optimization

This project presents an optimization model designed to support strategic decision-making in real estate investment.

Problem Identification

We identified the need for a data-driven approach to balance short-term sales revenue with long-term rental income in residential property investments. Focusing on the Serangoon area, our project aimed to assist firms in making optimal purchase, rent, and sell decisions under realistic constraints.

Model Development

We developed a mathematical model that incorporates real-world considerations, including purchase prices, rental yields, selling prices, transaction costs, agent commissions, and the present value of future income streams. The model was formulated to optimize total expected returns while considering time-based financial trade-offs.

Validation & Feasibility

Although simplified to handle unpredictable market variables, the model produced practical and financially sound recommendations. It provides a solid foundation for evaluating and optimizing investment portfolios based on data and projections.

Solution via Julia

Using the Julia programming language and optimization libraries, we implemented and solved the model using real and simulated datasets. The solution revealed the ideal quantity and type of property transactions to maximize profit across different investment timelines.
